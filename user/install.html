Installation¶

Dependencies¶

conda install numpy pymc3 astropy exoplanet

or

pip install numpy pymc3 astropy exoplanet

From Source¶

git clone --recursive https://github.com/danhey/maelstrom.git
cd maelstrom
python setup.py install
