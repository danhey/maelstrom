Maelstrom¶

maelstrom is a set of custom PyMC3 Models and solvers for modelling binary orbits through the phase modulation technique. This approach fully captures variations in a light curve caused by an orbital companion.

To install the current version:

git clone https://github.com/danhey/maelstrom.git
cd maelstrom
pip install -e .
